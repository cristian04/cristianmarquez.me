About me

I started with IT when I was 14 years old, building programs on QBasic and MSDOS.

I ran my own "startup" fixing computers and building cybers cafes when I was aged 16.

Later on I discovered Linux and I started building and maintaining servers. I played with OpenMosix and with my first Virtual Machines. (VirtualPC and QEMU)

I built software, I learn how to test it and how to automate it. I did Infrastructure automation, Continuous Integration and Continous Deployment pipelines.

I started using Public Cloud, mainly AWS and Google Cloud. I discovered Docker and Kubernetes.... I now focus on Cloud Computing, Cloud Native, DevOps and Site Reliability Engineering.
